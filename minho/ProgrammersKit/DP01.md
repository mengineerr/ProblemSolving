### How to Solve
- 최대 count가 8이므로 DFS 활용
- N 한도내에서 N, NN, NNN, .... 을 모두 사용하여 DFS

### dfs
  void dfs(int cur, int cnt){
      if(cnt >= 9)
          return;

      if(cur == Target){
          answer = min(answer,cnt);
          return;
      }

      int tmp = 0;

      for(int i = 0; i + cnt < 9; i++){
          tmp = tmp * 10 + Num;

          dfs(cur+tmp, cnt+1+i);
          dfs(cur-tmp, cnt+1+i);
          dfs(cur*tmp, cnt+1+i);
          dfs(cur/tmp, cnt+1+i);
      }
  }
