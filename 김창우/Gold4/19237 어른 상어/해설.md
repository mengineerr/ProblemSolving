**SHARK 구조체 [현위치, 다음위치, 우선순위, 생존여부]**
```
struct SHARK {
	int currentRow, currentCol, dir;
	int nextRow, nextCol;
	int prior[5][5];
	bool status; // true : alive, false : death
};
```

**장소물색 후 상어흔적을 계산**
```
void changeTrace(){
	for(int i = 1 ; i <= N ; i++){
		for (int j = 1 ; j <= N ; j++){
			if(trace[i][j] > 1) trace[i][j]--;
			else if(trace[i][j] == 1){
				trace[i][j] = 0;
				map[i][j] = 0;
			}
		}
	}
}
```
