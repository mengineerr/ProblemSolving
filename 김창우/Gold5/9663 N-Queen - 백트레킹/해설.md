```
bool check (int col){
	for (int i = 1 ; i < col ; i++){
		if(column[i] == column[col]) return false;
		if(abs(column[i] - column[col]) == abs(i - col)) return false;
	}
	return true;
}
```

**DFS하기전에 다음노드가 유망한지 체크한다.**

```
void DFS(int col){
	if (col > N){
		answer++;
	}else{
		for (int i = 1 ; i <= N ; i++){
			column[col] = i;
			if(check(col)) DFS(col + 1);
		}
	}
}
```
